# Job Analyzer

Find, analyze, and track job opportunities in the Israeli tech market.

## Features
- **Job Analysis**: 6-point scoring system (Match, Income, Growth, LowPrep, Stress, Location)
- **Application Tracking**: Track status via LinkedIn
- **Contact Management**: Keep track of recruiters
- **Follow-ups**: Automated reminders

1. `Analyze: [URL]`
2. `Track LinkedIn: [URL]`
