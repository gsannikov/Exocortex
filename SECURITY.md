## Reporting a Vulnerability

We take the security of this project seriously. If you discover a vulnerability, please report it immediately.

**Please do NOT open an issue for security vulnerabilities.**

Instead, please report vulnerabilities by:

1.  Emailing the project maintainer (see GitHub profile).
2.  Using the GitHub Security Advisory "Report a vulnerability" button (if enabled).

We will acknowledge your report within 48 hours and provide an estimated timeline for remediation.

### Policy

*   We request that you allow us reasonable time to address the vulnerability before publicly disclosing it.
*   We will provide full credit to reporters who follow this policy.
